# Expense Tracker

A simple command-line Python project for tracking expenses. The program lets the user add expenses, view saved expenses, calculate the total amount spent, delete expenses, and save the data using JSON.

## Features

- Add a new expense with a description, cost, and category
- View all saved expenses
- Calculate the total cost of all expenses
- Delete an expense
- Save expenses to a JSON file
- Load saved expenses when the program starts

## How To Run

From the `Expense_Tracker` folder, run:

```bash
python3 main.py
```

## Notes

The program saves expense data in `expenses.json`. This file is ignored by Git so personal expense data does not get uploaded to GitHub.
