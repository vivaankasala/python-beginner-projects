# Weather App

A simple command-line Python weather app that uses the OpenWeatherMap API to get live weather data for a city.

## Features

- Ask the user for a city name
- Send a request to the OpenWeatherMap API
- Read JSON data from the API response
- Print a clean weather summary
- Show temperature, feels-like temperature, condition, humidity, and wind speed
- Save the full API response to `weather.json`

## How To Run

From the `Weather_App` folder, run:

```bash
python3 main.py
```

## API Key Setup

This project uses a `config.py` file to keep the API key out of GitHub.

Create a file named `config.py` in the `Weather_App` folder:

```python
api_key = "YOUR_OPENWEATHERMAP_API_KEY"
```

The `.gitignore` file ignores `config.py`, so your real API key stays private.
