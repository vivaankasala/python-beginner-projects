# Python Beginner Projects

A collection of small Python projects built while learning programming,
project structure, virtual environments, Git, and browser automation.

## Projects

### Habit Tracker

A command-line program that can:

- Add habits
- Display saved habits
- Mark habits as complete
- Delete habits
- Save data in JSON

### WhatsApp Automation

A Selenium project that:

- Opens WhatsApp Web
- Reuses a private Chrome profile
- Searches for a contact
- Sends a message
- Waits for WhatsApp to confirm the message was sent

See each project's README for its setup and usage instructions.

## Privacy

Virtual environments, Chrome session data, environment files, and Python
cache files are excluded from Git.
